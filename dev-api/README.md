# PickProject - API

This project was bootstrapped with [Python Flask](https://flask.palletsprojects.com/en/2.0.x/).

## Available Scripts

In the project directory, you can run:

### `api start`

**Local**

python app.py

**Server**

nohup python3 app.py > output.log

### `api stop`

ps x | grep python3
kill PID (*check to PID)
